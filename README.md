# RAG-hybrid-retrieval-evaluator
An advanced Retrieval-Augmented Generation (RAG) system combining BM25, FAISS vector search, and cross-encoder reranking for high-precision document retrieval. Features LLM-based evaluation (accuracy, coverage, citation match) to ensure reliable, explainable answers. Built with Python, LangChain, and Streamlit for interactive Q&amp;A.
